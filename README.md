# Image Classification using CNN

This project is a deep learning model that classifies images of cats and dogs using Convolutional Neural Networks (CNN).

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy

## Features
- Image preprocessing
- CNN model training
- Image classification (Cat vs Dog)

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Train the model:
python train.py

3. Test prediction:
python predict.py

## Output
The model predicts whether the image is a Cat or Dog.
